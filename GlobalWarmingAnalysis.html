The Effect of COVID on Pollution Levels in India

Description: Our project looked at pollution levels in India before COVID and compared them to pollution levels during COVID and wanted to determine if the consequences of COVID had an effect on the degree of pollution in India.

Hypothesis: Pollution levels during COVID were lower than pollution levels before COVID due to reduced social interaction in communities.

We used the Pandas module and the NumPy module. The Pandas was used to organize the data frame and the NumPy module was used to perform mathematical operations on our data frame.

We dropped any row with null values from the data frame and took the average value across each date for each pollutant since each city recorded the amount of pollutant for each date.

We graphed the levels of each pollutant across the entire dataset to explore our data, which spanned from January 2015 to July 2020.

# Function that generates a Day Versus Pollutant Graph for any pollutant
# I.e. we can pass in NO as the input and it'll generate a Day Number Versus No Content graph

def generate_pollutant_graph(pollutant_name, pollutant_name_index, dataset):
    
    pollutant_values = [] # use square brackets instead of parentheses
    
    dataset_array = dataset.to_numpy()
    
    for i in range(len(dataset_array)): #the length of dataset_array
        
        pollutant_values.append(dataset_array[i][pollutant_name_index])
        
    plt.xlabel('Day Number')
    
    plt.ylabel(pollutant_name + ' Content (Per Day)')
    
    plt.title('Day Number Versus ' + pollutant_name + ' Content')
    
    plt.scatter(x = range(len(dataset_array)), y = pollutant_values)
    
    plt.show()
    
generate_pollutant_graph('NH3', 5, dataset)

We split up the data into two categories: the first category consisted of data that was recorded before COVID. The second category consisted of data that was recorded during COVID.

To do this, we just checked if the year was 2020. If so, we automatically added it to the dataset_before_covid. If the year was 2020, we checked if the month was before February and if it was, then we added that to the dataset_before_covid as well. Otherwise, we would add it to the dataset during COVID.

    date = dates[i]
    datapoint = dataset_array[i]
    
    if(int(date[:4]) < 2020):
        dataset_before_covid.append(datapoint)
    else:
        if (int(date[5:7]) < 2):
            dataset_before_covid.append(datapoint)

            dataset_during_covid.append(datapoint)

We wrote a function in which we could pass in a name of a chemical, its chemical formula, our before dataset, and our during dataset. The function would print the average value of the chemical before COVID and during COVID.

In [28]:
def print_before_and_after_covid(name, chemical_name, before_dataset, after_dataset):
    
    levels_before_covid = before_dataset[chemical_name]
    levels_during_covid = after_dataset[chemical_name]
    
    before_covid_mean = levels_before_covid.mean()
    during_covid_mean = levels_during_covid.mean()
    
    print("The average " + name + " content in the air before Covid was:")
    print(before_covid_mean)
    
    print("The average " + name + " content in the air during Covid was:")
    print(during_covid_mean)
    
print_before_and_after_covid('Ammonia', 'NH3', before_covid_dataframe, during_covid_dataframe)

def generate_line_of_best_fit(pollutant, pollutant_chemical_name, dataset):
    y_values = dataset[pollutant_chemical_name].to_numpy()
    x_values = numpy.asarray(list(range(len(y_values))))
    line_of_best_fit = LinearRegression()
    line_of_best_fit.fit(x_values.reshape(-1, 1), y_values.reshape(-1, 1))
